LGV_MeetingSDK_Meeting_Physical_Protocol

This defines the meeting’s physical location component.

  • placemark

    REQUIRED** The location is stored as a standard placemark.

  • locationCoordinates Default implementation

    OPTIONAL** This displays the coordinates (degrees Long/Lat). Nil, if no coordinates (address only).

    Default Implementation

    The default returns the coordinates directly from the placemark.
